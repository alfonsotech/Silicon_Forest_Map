Silicon Forest Map

Copyright 2016 Todd Brochu

An interactive map of all of the current companies in the Portland area that employ software developers, software engineers and / or web developers

Start by clicking on a region from the menu on the left side.
The map will zoom to the selected region.
Companies within the selected region will be displayed on the map as a marker as well as in the Companies list at the bottom left.
Each item in the Companies list contains a link to the website of that company.

Hovering over a marker on the map will display basic information for that company.
Clicking on a marker will zoom directly to the corresponding address.
To zoom back out to the previous view, simply click on the marker again.
